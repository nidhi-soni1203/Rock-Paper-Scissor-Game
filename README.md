# Rock Paper Scissors Game

An interactive Rock Paper Scissors game where users can compete against the computer. The game includes random computer choices, result calculation, and score tracking to provide an engaging user experience.

## Features
- Player vs Computer gameplay
- Random computer choice generation
- Real-time result display
- Score tracking system
- Responsive and user-friendly interface

## Technologies Used
- HTML5
- CSS3
- JavaScript

## Purpose
This project was developed to practice JavaScript concepts such as DOM manipulation, event handling, conditional logic, and random number generation.
